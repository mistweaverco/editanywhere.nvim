INTRODUCTION                                    *editanywhere*

This plugin's main purpose is to browse and edit EditAnywhere Server
ressources.

You have to have the [EditAnywhere Server](https://github.com/superevilmegaco/editanywhere-server)
installed and running.

You also need to have [curl](https://curl.haxx.se/) and
[jq](https://stedolan.github.io/jq/) installed.


COMMANDS                                        *editanywhere-commands*

editanywhere					      *:EditAnywhereBrowse*

        Browse and edit EditAnywhere Server ressources.


MAPPINGS                                        *editanywhere-mappings*

There are no mappings registered by the plugin.
You need to map it according to your preference.
